template<typename patchStringIteratorType , typename definitionStringIteratorType , typename replacementStringIteratorType >
void worldlib::replacePatchDefinition ( patchStringIteratorType  patchStart,
patchStringIteratorType  patchEnd,

definitionStringIteratorType  definitionEnd,
replacementStringIteratorType  valueToChangeToStart,
replacementStringIteratorType  valueToChangeToEnd 
)

Given a string to a patch, finds a definition (e.g. "!true = $00") and replaces it with a new string.

It's highly recommended to look at the version that just takes normal strings instead of iterators so that this function doesn't take up like 5 lines of back_insert_iterator boilerplate.

Parameters
patchStartAn iterator pointing to the patch's start
patchEndAn iterator pointing to the patch's end
definitionStartAn iterator pointing to a string containing the name of the definition you want to replace, not including the "!"
definitionEndAn iterator pointing to the definition's end
valueToChangeToStartAn iterator pointing to a string containing the name of the value you want to replace the original definition's value with
valueToChangeToEndAn iterator pointing to the value's end
Exceptions
std::runtime_errorThrown if:
  1. The definition cannot be found
  2. The definition is malformed
  3. The definition's value's length (in characters) is not equal of the length of the string you want to replace it with. E.G. trying to replace "!value = $0000" with "$50" instead of "$0050"

Given a string to a patch, finds a definition (e.g. "!true = $00") and replaces it with a number, printed as a hex value (with a $ preceding it).

It's highly recommended to look at the version that just takes normal strings instead of iterators so that this function doesn't take up like 5 lines of back_insert_iterator boilerplate.

Parameters
patchStartAn iterator pointing to the patch's start
patchEndAn iterator pointing to the patch's end
definitionStartAn iterator pointing to a string containing the name of the definition you want to replace, not including the "!"
definitionEndAn iterator pointing to the definition's end
valueToChangeToThe numeric value to use (will be converted to a hex value with a '$' preceding it)
digitsHow mny digits long the value should be (for example, use 4 to convert "$50" to "$0050")
Exceptions
std::runtime_errorThrown if:
  1. The definition cannot be found
  2. The definition is malformed
  3. The definition's value's length (in characters) is not equal of the length of the string you want to replace it with. E.G. trying to replace "!value = $0000" with "$50" instead of "$0050"

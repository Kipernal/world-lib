Detailed Description

Module for decompression functions. You'll probably want the more specific graphics decompression functions in the Level module instead.

Function Documentation

template<typename inputIteratorType , typename outputIteratorType >

inputIteratorType  compressedDataEnd,
outputIteratorType  out,
int *  compressedSize = nullptr,
int *  decompressedSize = nullptr

Decompresses data compressed in the LZ2 format.

In general you'll want the functions in level.hpp related to getting graphics files instead, but this may be useful if you have your own data compressed like this.

Parameters
compressedDataStartAn iterator pointing to the beginning of the compressed data
compressedDataEndAn iterator pointing to the end of the compressed data or any valid point after that (for example, the end of the ROM).
outWhere to output the data
compressedSizeWill contain the size of the compressed data after the function ends if it is not nullptr
decompressedSizeWill contain the size of the decompressed data after the function ends if it is not nullptr
Returns
Iterator pointing to the end of your decompressed data
Exceptions
std::runtime_errorAn error occurred while decompressing the data. Either there was an unrecognized bit sequence or there was not enough data to decompress
